## 1. 机制形式化定义

### 1.1 状态空间

```
S = {PROBE, CONFIRMED, TERMINAL_SUCCESS, TERMINAL_FAILURE}
```

- **PROBE:** 震荡试探阶段，仓位递增
- **CONFIRMED:** 趋势确认，大仓位持有或继续加仓
- **TERMINAL_SUCCESS:** 获得目标 R，重置
- **TERMINAL_FAILURE:** 达到风险预算上限，停止

### 1.2 仓位规则

**算术递增（固定金额风险）：**
```
r_n = min(r_0 + n × d, r_max)
```
- r_0: 初始风险金额（如账户的 1%）
- d: 递增金额（如账户的 0.5%）
- n: 连续止损次数（从 0 开始）
- r_max = r_0 + K × d: 最大仓位上限

**算术递增（权益比例风险）：**
```
r_n = min(r_0 + n × d, r_max) × W_{current}
```
- W_{current}: 当前账户权益
- 每次按当前权益计算风险金额

**几何递增（固定金额）：**
```
r_n = min(r_0 × m^n, r_max)
```
- m: 递增倍数（如 1.5 或 2.0）

**几何递增（权益比例）：**
```
r_n = min(r_0 × m^n, r_max) × W_{current}
```

### 1.3 转移规则

**PROBE 阶段：**
- 止损 → n += 1, 仓位递增到 r_{n+1}
- 若 n >= K → TERMINAL_FAILURE（达到最大档位）
- 获利 >= R_confirm × r_n → CONFIRMED
- 累计亏损 >= Budget → TERMINAL_FAILURE

**CONFIRMED 阶段：**
- 保持大仓位，或按 anti-martingale 加仓
- 获利达到总目标 → TERMINAL_SUCCESS
- 反转止损 → 根据规则可能回到 PROBE 或 TERMINAL

**TERMINAL 状态：**
- SUCCESS: 重置 n=0, 重新进入 PROBE
- FAILURE: 停止交易或等待冷却期

## 2. Break-Even 分析

### 2.1 固定金额风险 + 算术递增

前 n 次止损，每次亏损 r_i，累计亏损（忽略交易成本）：

**线性累积：**
```
L_n = Σ(i=0 to n-1) r_i 
    = Σ(i=0 to n-1) (r_0 + i × d)
    = n × r_0 + d × Σ(i=0 to n-1) i
    = n × r_0 + d × n(n-1)/2
    = n × (r_0 + d(n-1)/2)
```

第 n+1 次（n 次失败后的下一次）获胜，需要 R 满足：
```
R × r_n >= L_n
R >= L_n / r_n
R >= [n × (r_0 + d(n-1)/2)] / (r_0 + n × d)
```

**数值例子：**
- r_0 = 1%, d = 0.5%, K = 5
- n=0: L_0=0, r_0=1%, R>=0（首次）
- n=1: L_1=1%, r_1=1.5%, R>=0.67
- n=2: L_2=2.5%, r_2=2%, R>=1.25
- n=3: L_3=4.5%, r_3=2.5%, R>=1.8
- n=4: L_4=7%, r_4=3%, R>=2.33
- n=5: L_5=10%, r_5=3.5%, R>=2.86

**关键观察：**
- 所需 R 随 n **接近线性增长**，非二次或指数
- 但 r_n 也在增长，部分抵消
- 实际 R_min 从 0.67 增长到 2.86（~4.3倍）

### 2.2 固定金额风险 + 几何递增

前 n 次止损，仓位几何级数：
```
L_n = Σ(i=0 to n-1) r_0 × m^i
    = r_0 × (m^n - 1) / (m - 1)
```

第 n+1 次获胜需要：
```
R >= L_n / r_n
R >= [r_0 × (m^n - 1) / (m - 1)] / (r_0 × m^n)
R >= (m^n - 1) / [m^n × (m - 1)]
R >= (1 - m^(-n)) / (m - 1)
```

当 n → ∞, R → 1/(m-1)

**数值例子（m=2, doubling）：**
- n=1: L_1=r_0, r_1=2r_0, R>=0.5
- n=2: L_2=3r_0, r_2=4r_0, R>=0.75
- n=3: L_3=7r_0, r_3=8r_0, R>=0.875
- n=4: L_4=15r_0, r_4=16r_0, R>=0.9375
- n=5: L_5=31r_0, r_5=32r_0, R>=0.96875

**关键观察：**
- 所需 R 渐近趋向 1/(m-1) = 1.0（当 m=2）
- 看似"合理"，但仓位 r_n = 2^n × r_0 指数爆炸
- n=5 时仓位 = 32 × r_0（如 32% 权益），极高风险

**对比算术 vs 几何：**
- 算术：R_min 线性增长，仓位线性增长，较温和
- 几何：R_min 渐近收敛，但仓位指数增长，风险极高

### 2.3 权益比例风险

若每次亏损 r_i = (r_0 + i×d) × W_i，则权益递减：
```
W_{i+1} = W_i × (1 - r_i/W_i) = W_i - r_i
```

简化：仍按权益百分比，但权益在减少。

**几何复利效应：**
```
W_n = W_0 × Π(i=0 to n-1) (1 - ρ_i)
```
其中 ρ_i = (r_0 + i×d)/100 是权益百分比。

**累计亏损：**
```
L_n = W_0 - W_n = W_0 × [1 - Π(1 - ρ_i)]
```

第 n+1 次获胜，需要：
```
R × r_n >= L_n
R × (r_0 + n×d) × W_n >= W_0 × [1 - Π(1 - ρ_i)]
```

**数值例子：**
- r_0=1%, d=0.5%, W_0=100,000
- n=1: W_1=99,000, L_1=1,000, r_1=1.5%×99,000=1,485, R>=0.673
- n=2: W_2=97,515, L_2=2,485, r_2=2%×97,515=1,950, R>=1.274
- n=3: W_3=95,565, L_3=4,435, r_3=2.5%×95,565=2,389, R>=1.857
- n=4: W_4=93,176, L_4=6,824, r_4=3%×93,176=2,795, R>=2.441
- n=5: W_5=90,381, L_5=9,619, r_5=3.5%×90,381=3,163, R>=3.041

**对比固定金额 vs 权益比例：**
- 固定金额：R_min 增长较慢（2.86 @ n=5）
- 权益比例：R_min 增长较快（3.04 @ n=5）
- 权益比例因权益递减，后期仓位绝对值变小，需更高 R 补偿

## 3. 期望收益分析

### 3.1 单周期期望（i.i.d. 假设）

假设每次交易独立同分布，胜率 p，盈亏比 R。

第 n+1 次交易（已 n 次止损）：
```
E[G_{n+1}] = p × r_{n+1} × R - (1-p) × r_{n+1}
           = r_{n+1} × (p × R - (1-p))
           = r_{n+1} × (p × R - 1 + p)
           = r_{n+1} × (p × (R + 1) - 1)
```

**条件：** E[G_{n+1}] > 0 当且仅当 p × (R + 1) > 1

**关键问题：** 若 p 和 R 恒定，递增 r_{n+1} 只是**按比例放大期望收益和风险**，不改变夏普比率或几何增长率。

### 3.2 累计期望（完整周期）

考虑完整周期：前 n 次止损，第 n+1 次获胜。

**累计亏损：** L_n（见 2.1 和 2.2）

**第 n+1 次盈利：** r_{n+1} × R

**净收益：**
```
Net = r_{n+1} × R - L_n
```

**周期期望：**
```
E[Net | n losses then win] = P(n losses then win) × Net
                            = (1-p)^n × p × (r_{n+1} × R - L_n)
```

**总期望（所有可能 n）：**
```
E[Total] = Σ(n=0 to K) (1-p)^n × p × (r_{n+1} × R - L_n)
         + P(K+1 losses) × (-L_K - r_K)
```

其中最后一项是到达上限 K 仍止损的终端亏损。

**破产项：**
```
P(ruin) = (1-p)^(K+1)
Terminal loss = L_K + r_K
```

### 3.3 i.i.d. 假设下的结论

**若胜率 p 和盈亏比 R 恒定（止损不提供信息）：**

1. 递增仓位不改变单次交易的期望收益率（p×R - (1-p)）
2. 递增仓位重新分配风险到后期交易
3. 若早期小仓止损，后期大仓获胜，总收益可能更高
4. 但若早期小仓止损，后期大仓也止损，总亏损更大
5. **期望与方差、偏度的关系取决于完整假设：**
   - **算术期望**（单期或多期求和）：仓位路径可能改变总期望，因为 E[r_n × outcome_n] ≠ E[r_n] × E[outcome_n] 当两者相关时
   - **几何增长率**（长期复利）：在 i.i.d. 假设下，若仓位与未来收益独立、完整周期内风险预算固定，几何增长率主要由 Sharpe ratio 决定，仓位路径不改变长期增长率
   - **实际情况**：仓位递增策略隐含路径依赖和风险预算再分配，可能改变实际几何增长率

**关键争议点：**
- 若 p 和 R 恒定，递增仓位是否只是制造正偏度，而非提高期望？
- 答案：**复杂**，取决于度量标准：
  - 单次交易期望收益率不变
  - 完整周期的算术期望可能因仓位路径改变
  - 几何增长率在理想 i.i.d. 假设下不变，但实际交易中路径依赖、风险预算约束、Kelly 偏离会影响结果
  - 递增确实改变分布形状（正偏度），但正偏度本身不保证几何增长改善

### 3.4 条件概率假设（regime 信息）

**假设：** 止损提供 regime 信息，p_{n+1} > p 或 R_{n+1} > R

**模型 A：Hazard rate 递增**
```
p_{n+1} = p_0 + α × n
```
即连续 n 次止损后，下一次获胜概率上升。

**模型 B：Regime 切换**
```
市场在震荡（胜率 p_low）和趋势（胜率 p_high）间切换
连续止损 → 增加"即将进入趋势"的后验概率
```

**若条件概率确实上升：**
```
E[G_{n+1} | n losses] = r_{n+1} × (p_{n+1} × (R + 1) - 1)
```
当 p_{n+1} > p_0 时，期望收益上升。

**此时递增仓位有理论价值：**
- 在低胜率时期用小仓位（降低损失）
- 在高胜率时期用大仓位（放大盈利）
- **前提：** 必须有可靠方法推断 regime 或条件概率

**关键问题：** 止损序列是否真的提供 regime 信息？需要实证检验。

## 4. 方差与偏度分析

### 4.1 方差

固定仓位策略：
```
Var[G] = r^2 × Var[outcome]
       = r^2 × [p × R^2 + (1-p) × 1 - (p×R - (1-p))^2]
```

递增仓位策略：
```
Var[Total] 取决于仓位路径的分布
```

**一般性质：**
- 递增仓位增加方差（后期大仓位波动更大）
- 若大仓位在高胜率时期，方差增加但期望也增加
- 若大仓位在恒定胜率时期，方差增加而期望不变（夏普比率下降）

### 4.2 偏度

**正偏度（小亏多次、大赢一次）：**
```
Skew > 0
```
- 多数路径：连续小额止损
- 少数路径：大仓位获胜，巨额盈利

**偏度来源：**
1. 仓位递增：后期仓位大
2. 盈亏比 R：获胜时盈利 = R × 止损
3. 概率分布：(1-p)^n × p，小概率大盈利

**与 Trend Following 相似：**
- 低胜率（30-40%）
- 高盈亏比（2-5）
- 正偏度分布

**偏度的价值：**
- 正偏度本身不保证正期望（见彩票）
- 但若 E > 0 且 Skew > 0，可能有心理优势（小亏可忍受）
- 风险：负偏度风险（如 Carry crash）更危险

## 5. Risk of Ruin 分析

### 5.1 到达最大档位 K 的概率

连续 K+1 次止损（从 n=0 到 n=K 全部失败）：
```
P(ruin) = (1 - p)^(K+1)
```

**数值例子：**
- p=0.4, K=5: P(ruin) = 0.6^6 = 4.67%
- p=0.3, K=5: P(ruin) = 0.7^6 = 11.76%
- p=0.4, K=10: P(ruin) = 0.6^11 = 0.36%

**关键观察：**
- K 越大，破产概率越低（更多尝试机会）
- 但 K 越大，到达 K 时的累计亏损越大
- Trade-off：破产概率 vs 最大亏损幅度

### 5.2 累计亏损分布

到达 K 时的累计亏损（算术递增）：
```
L_K = K × (r_0 + d(K-1)/2)
```

**数值例子：**
- r_0=1%, d=0.5%, K=5: L_K=10%
- r_0=1%, d=0.5%, K=10: L_K=27.5%
- r_0=2%, d=1%, K=5: L_K=24%

**与总风险预算对比：**
- 若 Budget=15%, K=10 会超出
- 必须设置 K 使 L_K <= Budget

### 5.3 Budget 约束下的最优 K

给定 Budget B，最大 K 满足：
```
L_K <= B
K × (r_0 + d(K-1)/2) <= B
```

解二次不等式：
```
K^2 × d/2 + K × (r_0 - d/2) - B <= 0
K <= [-（r_0 - d/2) + sqrt((r_0 - d/2)^2 + 2×d×B)] / d
```

**数值例子：**
- r_0=1%, d=0.5%, B=10%: K_max ≈ 5.7 → K=5
- r_0=1%, d=0.5%, B=15%: K_max ≈ 8.2 → K=8
- r_0=2%, d=1%, B=15%: K_max ≈ 4.2 → K=4

**Trade-off 可视化：**
- 小 K：低累计亏损，但高破产概率
- 大 K：低破产概率，但高累计亏损
- Budget 约束限制 K 上界

## 6. 与经典策略的数学关系

### 6.1 固定仓位（d=0）

特例：d=0, r_n = r_0 恒定

```
L_n = n × r_0
R_min = n × r_0 / r_0 = n
```

第 n 次失败后，需 R >= n 才能回本。

**对比本机制：**
- 固定仓位：R_min 线性增长，与仓位成正比
- 递增仓位：R_min 增长较慢（因为 r_n 也在增长）

### 6.2 Martingale（m=2, 无限）

几何递增，m=2，无上限 K：
```
r_n = r_0 × 2^n
L_n = r_0 × (2^n - 1)
R_min → 1（渐近）
```

**理论上** R>1 即可回本，但：
- 仓位 2^n 指数爆炸
- 有限资本下，大 n 时资本不足
- P(ruin) → 1（在有限资本约束下）

**本机制区别：**
- 有上限 K（有限尝试）
- 算术递增可选（非指数）
- 总预算约束

### 6.3 Anti-Martingale（盈利递增）

盈利后增加仓位，止损后减少或保持。

**数学上相反：**
- Anti: 高胜率时用大仓位
- 本机制: 止损序列后用大仓位（若止损提供信息）

**若止损不提供信息，本机制接近 Martingale 风险特征。**

### 6.4 Kelly Criterion

Kelly 最优仓位（二项模型）：
```
f* = (p × (R+1) - 1) / R = (p × R - q) / R
```

**应用到本机制：**
- 可用 Kelly 计算每档仓位：r_n = f* × W_n
- 若递增超过 Kelly，over-betting 风险
- 若递增低于 Kelly，under-betting（可能错失增长）

**递增与 Kelly 的冲突：**
- Kelly 基于固定 p 和 R
- 递增假设 p 或 R 在变化（或路径依赖）
- 若 p 恒定，Kelly 推荐恒定 f*，非递增

## 7. 关键数学结论

### 7.1 Break-even R 的增长

**算术递增：**
- R_min 增长率：接近线性，略低于 O(n)
- 可控，但仍需 R > 1 且随 n 递增

**几何递增：**
- R_min 渐近收敛到 1/(m-1)
- 看似"低要求"，但仓位指数爆炸
- 实际上是 Martingale 风险

**权益比例 vs 固定金额：**
- 权益比例：权益递减，R_min 增长更快
- 固定金额：R_min 增长较慢，但后期仓位可能超出权益承受

### 7.2 期望收益的条件

**i.i.d. 假设（p, R 恒定）：**
- 递增仓位不改变单次交易的期望收益率
- 改变方差和偏度分布
- 制造正偏度（小亏多次、大赢少次）
- **在完整周期、风险预算固定、仓位与未来收益独立的理想条件下**，几何增长率主要由 Sharpe 决定，仓位路径的独立价值有限
- **但实际情况中**：路径依赖、Kelly 偏离、风险预算动态调整可能使结果偏离理想 i.i.d. 预测

**条件概率假设（p 或 R 递增）：**
- 若止损真的提供 regime 信息
- 递增仓位可能有期望价值
- **前提：** 必须实证验证止损的信息含量

### 7.3 风险特征

**方差：**
- 递增仓位增加方差
- 除非高胜率时期，否则夏普比率可能下降

**偏度：**
- 正偏度（小亏多、大赢少）
- 类似 trend following 特征
- 正偏度本身不保证正期望

**Risk of Ruin：**
- P(ruin) = (1-p)^(K+1)
- K 越大，破产概率越低
- 但累计亏损 L_K 越大
- 必须在 Budget 约束下选择 K

### 7.4 与经典策略的位置

**连续谱：**
```
固定仓位 ←→ 有限算术递增 ←→ 有限几何递增 ←→ 无限 Martingale
(d=0)         (d>0, K有限)      (m>1, K有限)      (m>1, K→∞)
```

**本机制：**
- 介于固定仓位和有限 Martingale 之间
- 通过 K 和 Budget 控制风险
- 算术递增比几何递增温和

**关键区别：**
- 本机制 ≠ 无限 Martingale（有上限和预算）
- 本机制 ≠ Anti-Martingale（触发条件相反）
- 本机制类似有限、有预算的 Martingale 变种

### 7.5 增量价值来源（理论上）

**若要有增量价值，必须满足至少一个：**

1. **止损提供 regime 信息：** p_{n+1} > p_0
2. **假突破聚集在趋势前：** 连续止损增加趋势概率
3. **条件盈亏比改善：** R_{n+1} > R_0
4. **心理/执行优势：** 小仓试错降低心理压力，更易执行

**若以上均不成立：**
- 递增仓位主要重新分配风险路径
- 制造正偏度，但在理想 i.i.d. 条件下不改善几何增长率
- 可能因方差增加而降低夏普比率
- **结论：在理想 i.i.d. 假设下增量价值有限，且可能因偏离 Kelly 而有害；实际效果需实证检验**

### 7.6 最危险的误区

1. **"必然回本"谬误：**
   - 即使 R_min 计算正确，不保证下次获胜
   - P(n+1 次仍失败) = (1-p)^(n+1) > 0

2. **"指数盈利"谬误：**
   - 财富过程仍是乘法的
   - 递增仓位不改变几何增长的本质
   - 只改变收益分布的偏度

3. **"震荡必然结束"谬误：**
   - 震荡可能持续超过 K 次
   - 趋势可能不出现
   - 或趋势出现但立即反转

4. **忽略交易成本：**
   - 每次试探有成本 c
   - n 次累计 n × c
   - R_min 必须加上成本：R_min = (L_n + n×c) / r_n

5. **忽略 Gap 和滑点：**
   - 止损可能无法执行于预期价格
   - 实际亏损 > r_n
   - 递增仓位放大 gap 风险

## 8. 数学模型总结

### 8.1 核心公式

**算术递增累计亏损：**
```
L_n = n × (r_0 + d(n-1)/2)
```

**几何递增累计亏损：**
```
L_n = r_0 × (m^n - 1) / (m - 1)
```

**Break-even R（算术）：**
```
R_min = L_n / r_n = [n × (r_0 + d(n-1)/2)] / (r_0 + n×d)
```

**Break-even R（几何）：**
```
R_min = (1 - m^(-n)) / (m - 1) → 1/(m-1) as n→∞
```

**Risk of Ruin：**
```
P(ruin) = (1 - p)^(K+1)
```

**Budget 约束最大 K（算术）：**
```
K_max = [-（r_0 - d/2) + sqrt((r_0 - d/2)^2 + 2×d×B)] / d
```

### 8.2 参数敏感性

| 参数 | 增加影响 |
|------|---------|
| r_0 ↑ | L_n ↑, R_min 变化小, 破产概率 ↓（若 p 恒定） |
| d ↑ | L_n ↑, R_min ↑, K_max ↓ |
| K ↑ | 破产概率 ↓, 最大亏损 L_K ↑ |
| p ↑ | 破产概率 ↓↓, 期望 ↑ |
| R ↑ | 回本更容易, 期望 ↑ |
| Budget ↑ | K_max ↑, 允许更多尝试 |

### 8.3 最佳实践（纯数学）

**若 p 和 R 已知且恒定：**
- 使用固定仓位或 Kelly，不递增
- 递增无期望价值

**若相信止损提供信息：**
- 使用算术递增（非几何）
- 设置合理 K（如 3-5）
- Budget 控制总风险（如 10%）
- 监控实际 p_{n+1} 是否确实上升

**若不确定：**
- 默认假设 i.i.d.
- 优先固定仓位或标准 sizing 方法
- 递增仓位作为假设，需实证检验
