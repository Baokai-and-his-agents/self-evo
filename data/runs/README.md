# Runs

Run summaries and optional JSONL event logs.

Recommended layout:

```text
data/runs/YYYY-MM-DD/<run-id>.summary.md
data/runs/YYYY-MM-DD/<run-id>.jsonl
```

Each run summary should include:

- GitHub Issue
- worker identity
- run id
- goal
- actions taken
- sources used
- files changed
- blockers
- next steps
