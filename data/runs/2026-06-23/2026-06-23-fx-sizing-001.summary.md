本研究针对"有限递增试探仓位"机制进行深入分析：震荡期小仓试探并止损，止损后在上限和预算内逐级提高仓位，趋势建立后由大仓获得高 R 收益。研究通过严格串行证据检索（Query 1-9）、数学模型推导、失效模式分析，明确回答：该机制是否有理论基础和实证支持？

**核心结论：**
1. **直接支持证据不存在**（Query 1-9 一致阴性）
2. **核心假设未验证**（止损序列不预测趋势，Query 9）
3. **若假设不成立，退化为有限 Martingale**（继承 fat tail 风险）
4. **主流共识完全相反**（止损 → 减仓，盈利 → 增仓）
5. **风险评级：极高**，不推荐实盘使用

## 核心发现

### 1. 实证状态（跨 Query 1-9 一致）

| 维度 | 发现 | 证据层级 |
|------|------|---------|
| 直接证据 | 止损后递增的支持证据**不存在** | N/A |
| Sequential testing | SPRT 应用于 change-point detection，未见仓位递增 | Tier 1-2 |
| Change-point | 识别后标准反应：减仓或策略切换 | Tier 2-3 |
| HMM regime | 高波动/bear → 减仓（25-10%），低波动/bull → 增仓（100%） | Tier 3-4 |
| Martingale 实证 | 所有来源强烈反对，fat tail risk，账户寿命 1-2 年 | Tier 3-4 |
| Turtle trading | 盈利后加仓（0.5N step），1983-1987 实盘 80%+ 年化 | Tier 2-3 |
| False breakout | 50-70% breakout 为 false，**不预测趋势** | Tier 4 |

**主流共识（100% 一致）：**
- 止损/亏损/drawdown → **减仓**
- 盈利/趋势确认 → **增仓**（anti-martingale）
- 高波动/不确定 → **减仓**
- 止损后增仓 → **不存在**

### 2. 数学模型关键洞察

**算术递增（r_n = r_0 + n×d）：**
- 累计亏损：L_n = n × (r_0 + d(n-1)/2)
- Break-even R：R_min 线性增长（0.67 → 2.86 @ n=5）
- 温和但仍需 R 随 n 递增

**几何递增（r_n = r_0 × m^n）：**
- 累计亏损：L_n = r_0 × (m^n - 1) / (m - 1)
- Break-even R：R_min → 1/(m-1)（渐近收敛）
- 看似"低要求"，但仓位指数爆炸（r_5 = 32×r_0）

**i.i.d. 假设下：**
- 若 p 和 R 恒定，递增仅重新分配风险，在理想条件下对几何增长率的独立价值有限
- 实际效果受路径依赖、Kelly 偏离、风险预算动态调整影响

**条件概率假设：**
- 若止损后 p 或 R 上升，递增可能有理论价值
- **前提：必须实证验证**（Query 9 不支持）

### 3. 风险评估

| 维度 | 评分 | 说明 |
|------|------|------|
| 理论风险 | **极高** | 核心假设（止损序列预测趋势）未验证 |
| 市场风险 | **高** | 震荡市场 30-40% 时间失效 |
| Fat tail 风险 | **高** | Martingale 特征：Kurtosis 88.03 |
| 极端事件风险 | **高** | Gap/滑点可能致命（2015 Swiss Franc） |
| **综合评级** | **极高** | **不推荐实盘使用** |

**关键失效模式：**
- F1. 核心假设不成立（概率 >70%）→ 退化为有限 Martingale
- F2. Fat Tail 风险（每 10-20 周期）→ 小概率巨额亏损
- F4. 持续震荡（30-40% 时间）→ 无趋势出现
- F6. Gap/滑点（低概率但致命）→ 止损执行失败

### 4. 实证支持的替代方案

| 策略 | 实证验证 | 证据层级 |
|------|---------|---------|
| Kelly Criterion | Thorp 19.1% 年化（1969-1988），无亏损年 | Tier 1 |
| Turtle Pyramiding | 80%+ 年化（1983-1987 实盘） | Tier 2-3 |
| Anti-Martingale | Query 7 对比，趋势中优于 Martingale | Tier 3 |
| HMM Regime Sizing | 多来源工程实践，NIFTY 50 Sharpe 1.05 → 0.67 | Tier 3-4 |
