# ChatGPT 会回答问题，AI Agent 为什么能替你办事？

> 状态：初稿，待人工审阅  
> 读者：听说过 ChatGPT、不了解 AI Agent 的普通用户  
> 核心观点：聊天机器人主要给答案，Agent 围绕目标持续采取行动；能行动不等于可以越过人的授权。

假设你准备去成都玩三天。

你问 ChatGPT：“帮我做一份成都旅行攻略。”

它很快给你一份清单：宽窄巷子、熊猫基地、火锅、交通建议。这个回答可能很有用，
但接下来你还是要自己查天气、比较酒店、调整时间、把行程放进日历，还要在价格
变化后重新安排。

如果换成一个 AI Agent，理想情况下，你给它的不是一道问题，而是一个目标：

> 帮我准备一次三天的成都旅行，总预算 5000 元，不坐红眼航班，每天不要安排得太满。

它会把目标拆成步骤：查交通、比较住宿、看天气、安排路线、检查预算，再把结果整理
成一份待确认的计划。条件变化时，它还可以重新计算，而不是等你从头再问一次。

这就是最简单的区别：

**ChatGPT 更像一个随问随答的顾问；AI Agent 更像一个拿着任务清单、会使用工具的助理。**

## “会回答”和“会办事”，中间差了什么？

一个能办事的 Agent，通常多了四样东西。

第一，它有一个持续目标。

聊天结束后，普通问答往往也就结束了。Agent 需要记住“这件事还没完成”，知道现在
做到哪一步、还有什么没做。

第二，它会使用工具。

它可能需要搜索网页、读取你交给它的文件、计算预算、生成表格，或者在获得允许后
调用日历。没有工具的 Agent，就像一个很聪明但没有电脑和电话的助理，只能给建议，
不能真的推进事情。

第三，它会根据结果调整下一步。

如果理想酒店满房，它应该回到候选列表继续比较；如果预算超了，它要知道该调整交通、
住宿还是行程，而不是把一个明显超支的方案照样交给你。

第四，也是最重要的，它必须有边界。

“帮我比较三家酒店”和“替我付款预订”不是同一件事。前者是准备信息，后者会花钱、
产生合同和取消成本。一个可靠的 Agent 应该在关键动作前停下来，请你确认。

## Agent 不是住在电脑里的“另一个人”

我们很容易把 Agent 想成一个数字员工：它理解你、记得一切，还会主动把事情做完。

现实没有这么整齐。

AI 可能误解目标，可能在看起来很自信时用错信息，也可能把一个小错误带到后面的每个
步骤。任务越长、使用的工具越多，出错的地方也越多。

所以，“能自己决定下一步”只是 Agent 的能力，不是它天然值得信任的证明。

真正有用的 Agent，不是永远不需要人，而是知道什么时候可以继续，什么时候必须停下。

## 普通人什么时候真的需要 Agent？

不是每个问题都值得使用 Agent。

如果你只想知道“成都三月穿什么”，一次问答就够了。硬要让 Agent 制订计划、调用工具，
反而更慢、更复杂。

Agent 更适合下面这类任务：

- 需要连续完成多个步骤；
- 中间结果会影响下一步；
- 需要重复执行，例如每周整理一次；
- 结果容易检查，出错后可以撤回；
- 关键决定可以留给人确认。

比如整理一周的会议记录、比较多个公开方案、把零散资料变成待办清单，都比“替我做
所有决定”更适合作为起点。

## 你今天就能做的一个小实验

不要一开始就把邮箱、银行卡或全部文件交给 AI。

先选一件低风险、可检查的任务，例如：

> 我会给你三份旅行资料。请整理成一份三天行程，标出冲突、未知信息和需要我决定的
> 三个问题。不要预订或发送任何东西。

这个任务已经带有 Agent 思维：有目标、有材料、有输出，也有明确边界。

未来真正改变工作的，也许不是一个无所不能的 AI，而是我们逐渐学会：怎样把任务说清楚，
怎样给工具，怎样检查结果，以及怎样保留最后那一下确认。

这时候，AI 才不只是回答问题，而是开始帮你把事情往前推进。
